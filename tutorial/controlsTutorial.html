Object of the game:

Goal
Description

In this game, shapes composed of blocks will be falling down the screen.

Your job is to move and rotate them into positions so that you get a completely full layer or plane. Think of it as a building, and you're trying to cover a single story.

When a shape can not fall any further down, you will have one last period of time to move it before it gets locked in place.

The Game ends when upon trying to insert a new shape in the top levels, it runs into a space that is already occupied.

Option Button Functions

Default

Description

Keyboard Trigger: Shift Key

This button switches how the shape responds to directions input on the control pad.
Pressing the circular arrow will allow the shape to then be rotated.

Keyboard Trigger: Shift Key

Pressing the crossed-arrows restores the shape to regular, directional movement.
